# pwa-library
PWA built with Ionic Angular and Firebase


## Developing

### First, be sure to install dependencies:

```
npm install 
```

### Then start a dev server on localhost:

```
ionic serve 
```

## Built With

* [Ionic 2]- The hybrid app framework used
* [Firebase] - Backend
